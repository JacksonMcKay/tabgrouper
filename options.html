Rules

How to write rules

One rule per line.

Rules follow the format PATTERN TAB_GROUP_NAME TAB_GROUP_COLOR.

Color is optional.

Pattern is a domain name and optional path. A domain and any subdomains will match, and if you provide a path, then the URL must start with that path also.

Lines beginning with # are comments, and blank lines are skipped.

Valid colors: grey, blue, red, yellow, green, pink, purple, cyan. Invalid colors are treated as if you didn't specify a color.

Examples

example.com Example pink - Will group example.com, www.example.com, foo.example.com in a tab group called "Example", that is colored pink.

example.com/foo ExampleFoo - Will group example.com/foo, www.example.com/foo, and example.com/foo/anything together but not example.com or example.com/bar
